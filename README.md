# 라즈베리파이를 이용한 센서 프로젝트

초음파센서에 10cm내에 물체가 탐지되면 부저가 울리고 led가 켜지게 설정함

## 사용센서 리스트

초음파센서(HC-SR04), LED(Dual Color LED), 부저(Active Buzzer Module)
